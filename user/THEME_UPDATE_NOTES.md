# Steve Jon Blue Theme Update

Theme updated to match the supplied Steve Jon Solutions LLP launch poster.

Palette used:

- Primary blue: `#0077FF`
- Primary hover: `#005ED1`
- Navy text: `#061B3A`
- Deep navy sections: `#021631`
- Secondary navy: `#002B66`
- Cyan glow/accent: `#35D8FF`
- Page background: `#F5FAFF`
- Soft surface blue: `#E7F2FF`

Main files changed:

- `src/app/globals.css` – added blue theme tokens and global page background.
- `src/app/layout.tsx` – metadata and base body colors.
- Major app pages and homepage components – replaced old cream/gold/black palette with blue/navy/cyan theme.

Note: this is a visual theme conversion only. The project still contains fashion/atelier copy in many sections. Rewrite the content separately if this site is being converted to a software/cloud/cybersecurity consulting company website.
